## Prerequisites
 - Git
 - Yarn
 - Node.js
 - NPM


## Getting Started

- git clone https://github.com/fegzycole/Private-Events.git

- cd Private-Events

- rails db:migrate (to add tables and columns)

- rails server to fire the rails server
